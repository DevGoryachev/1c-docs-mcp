# Tools v1

## 1. list_topics
Возвращает список тем и количество документов по каждой теме.

## 2. search
Параметры:
- query
- topic (опционально)
- top_k

Возвращает:
- query
- topic
- top_k
- total
- items

## 3. fetch
Параметры:
- id

Возвращает:
- id
- found
- item (`null`, если документ не найден; полный объект документа, если найден)
