# Темы документации

Используем только эти topic-коды:

- forms
- client_server
- interface
- http_services
- json
- odata
- web_services
- skd
- queries
- performance
- security
- team_rules

## Расшифровка

### forms
Формы, элементы формы, команды формы, поведение UI

### client_server
Границы клиента и сервера, вызовы сервера, директивы, контекст

### interface
Общий интерфейс, разделы, команды, такси, навигация

### http_services
HTTP-сервисы, REST, ответы, коды ошибок

### json
Чтение/запись JSON, сериализация, структура обмена

### odata
Автоматический REST-интерфейс OData

### web_services
Web-сервисы, SOAP, внешние сервисы

### skd
Система компоновки данных, параметры, группировки, ресурсы, настройки

### queries
Язык запросов, временные таблицы, структура запросов

### performance
Оптимизация, минимизация вызовов, тяжёлые операции

### security
Права, роли, ограничения доступа

### team_rules
Внутренние правила команды и проекта
